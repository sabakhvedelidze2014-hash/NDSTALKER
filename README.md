## 📖 პროექტის აღწერა / PROJECT DESCRIPTION

**GE:** **NDSTALKER** არის პითონზე დაწერილი ავტომატიზებული ხელსაწყო, რომელიც შექმნილია ღია წყაროებიდან ინფორმაციის სწრაფად მოძიებისთვის (OSINT). პროგრამა აერთიანებს მომხმარებლის მიერ შეყვანილ მონაცემებს (სახელი, გვარი, ასაკი, ტელეფონი) და ახდენს მათ ოპტიმიზებულ ძიებას Google-ის სისტემაში.

**EN:** **NDSTALKER** is a Python-based automation tool designed for quick Information Gathering from open sources (OSINT). The program combines user-inputted data (Name, Last Name, Age, Phone) and performs an optimized search through Google.

## 🚀 ფუნქციები / FEATURES

* ✅ **Automated Search:** ავტომატური ძებნა ყველა პარამეტრით.
* ✅ **Colorful UI:** ფერადი და მიმზიდველი ინტერფეისი ტერმინალში (Colorama).
* ✅ **Cross-Platform:** მუშაობს Windows, Linux და macOS სისტემებზე.
* ✅ **Safe Execution:** იყენებს `webbrowser` მოდულს უსაფრთხოებისათვის.
* ✅ **ASCII Art:** ლამაზი ვიზუალური ეფექტები გაშვებისას.
